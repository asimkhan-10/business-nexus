# Business Nexus

Full-stack MERN starter with role-based auth, profiles, collaboration requests, and real-time chat.

## Quickstart

### Local
1. Backend
   ```bash
   cd server
   cp .env.example .env
   npm install
   npm run dev
   ```
2. Frontend
   ```bash
   cd client
   cp .env.example .env
   npm install
   npm run dev
   ```

### Docker
```bash
docker compose up --build
```
